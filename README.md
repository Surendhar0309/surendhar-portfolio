# Surendhar V — Portfolio

> Personal portfolio website built with pure HTML, CSS, and JavaScript.  
> Live at: `https://Surendhar0309.github.io/surendhar-portfolio`

---

## About

A single-file, zero-dependency portfolio showcasing my work as a Data Science & ML graduate student. Features a dark navy theme with cyan glow accents and horizontal scroll navigation across 7 sections.

---

## Sections

- **Hero** — Introduction, key stats, and tech stack
- **About** — Background, education, and highlights
- **Skills** — Technical skills across Data Science, ML, Visualization, and Dev
- **Projects** — Featured projects with descriptions and links
- **Experience** — Internships and academic timeline
- **Certifications** — 6 professional certifications
- **Contact** — Email, LinkedIn, GitHub, and Hugging Face links

---

## Features

- Horizontal scroll navigation (mouse wheel, arrow keys, touch swipe)
- Custom glowing cursor with smooth lag ring
- Animated progress dots and section transitions
- Fully responsive — falls back to vertical scroll on mobile
- Zero frameworks, zero dependencies — pure HTML/CSS/JS

---

## Tech Stack

```
HTML5 · CSS3 · Vanilla JavaScript
Fonts: Syne + Inconsolata (Google Fonts)
```

---

## Local Setup

No build step needed. Just open the file in a browser:

```bash
git clone https://github.com/Surendhar0309/surendhar-portfolio.git
cd surendhar-portfolio
open index.html
```

---

## Deployment

Hosted via **GitHub Pages**:

1. Push `index.html` to the `main` branch
2. Go to **Settings → Pages → Source → main → / (root)**
3. Site goes live at `https://Surendhar0309.github.io/surendhar-portfolio`
