## Overview

This portfolio presents my background, skills, projects, experience, and certifications through a clean off-white interface with warm serif typography. It uses smooth anchor-scroll navigation with scroll-reveal animations and a fully responsive layout — designed for recruiter and collaborator audiences.

---

## Features

| Feature | Description |
|---|---|
| Smooth Scroll Navigation | Anchor-based navigation with fade-up reveal animations on scroll |
| Fully Responsive | Adapts cleanly across desktop, tablet, and mobile screen sizes |
| Scroll Reveal Animations | IntersectionObserver-powered animations — no scroll event listeners |
| Clean Typography | DM Serif Display paired with DM Sans for a professional editorial tone |
| Zero Dependencies | Pure HTML, CSS, and JavaScript — no npm, no build step required |
| Fast Load | Lightweight structure with minimal overhead and optimized font loading |

---

## Sections

1. **Hero** — Name, role, description, key stats, and tech stack

## Getting Started

No installation or build step is required.

**Clone the repository:**

```bash
git clone https://github.com/Surendhar0309/surendhar-portfolio.git
cd surendhar-portfolio
```

**Open in browser:**

```bash
open index.html
```

All three files — `index.html`, `style.css`, and `script.js` — must remain in the same directory to work correctly.

---

## Deployment

### GitHub Pages

1. Push all files to the `main` branch of your repository
2. Navigate to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Click **Save**

The site will be live at:

```
https://Surendhar0309.github.io/surendhar-portfolio
```

### Netlify

1. Visit [netlify.com](https://netlify.com) and sign in
2. Drag and drop the project folder onto the Netlify dashboard
3. The site goes live instantly at a free subdomain

## Performance

- No external JavaScript libraries — reduces initial load time significantly
- Google Fonts loaded with `rel="preconnect"` for faster font fetching
- CSS custom properties used throughout for consistent theming without repetition
- IntersectionObserver handles all scroll animations — no continuous scroll event listeners
- Minimal DOM depth and lightweight markup across all sections
